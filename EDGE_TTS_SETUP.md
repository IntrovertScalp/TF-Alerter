# Установка edge-tts для отличных голосов

## Быстрая установка

```powershell
# В корне проекта
.venv\Scripts\pip.exe install edge-tts
```

## Использование

1. Запустите программу
2. Откройте Настройки
3. В разделе "Фандинг: звук и голос":
   - TTS движок: выберите "Edge TTS (онлайн)"
   - Язык: выберите "Русский" или "English"
   - Голос: выберите любой из списка
4. Нажмите ▶ для теста
5. Сохраните настройки

## Доступные русские голоса Edge TTS

- **Dmitry** - мужской, естественный (РЕКОМЕНДУЕТСЯ)
- **Svetlana** - женский
- **Dariya** - женский

## Доступные английские голоса

- **Guy** - мужской (РЕКОМ мендуется)
- **Aria** - женский
- **Davis** - мужской
- **Jane** - женский

## Примечания

- Требуется интернет соединение
- Отличное качество речи
- Полностью бесплатно
- Поддержка 100+ языков и голосов
