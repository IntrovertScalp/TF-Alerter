====================================================================
ДОПОЛНИТЕЛЬНЫЕ ГОЛОСОВЫЕ ПАКИ ДЛЯ ФАНДИНГА
====================================================================

Эта папка предназначена для хранения альтернативных голосовых
паков для алертов фандинга.

Создайте подпапки с именами голосовых паков, например:
- BT_Titanfall/
- Custom_Male/
- Custom_Female/
- Robotic/

В каждой подпапке положите голосовой файл, например:
- funding_alert.wav
