====================================================================
ГОЛОСОВЫЕ ФАЙЛЫ ДЛЯ АЛЕРТОВ ФАНДИНГА
====================================================================

В этой папке должны находиться предзаписанные голосовые файлы
для алертов фандинга. Программа будет проигрывать выбранный файл
вместо синтеза речи через TTS.

--------------------------------------------------------------------
ТРЕБОВАНИЯ К ФАЙЛАМ:
--------------------------------------------------------------------
- Формат: .wav, .mp3, или .ogg
- Рекомендуется: 16-bit, 44100 Hz WAV
- Длительность: 1-3 секунды
- Язык: русский или английский

--------------------------------------------------------------------
ПРИМЕРЫ ИМЕН ФАЙЛОВ:
--------------------------------------------------------------------

funding_alert_ru_male.wav           - Русский мужской голос
funding_alert_ru_female.wav         - Русский женский голос  
funding_alert_en_male.wav           - Английский мужской голос
funding_alert_en_female.wav         - Английский женский голос

funding_alert_ru_titanfall.wav      - Стиль Titanfall (русский)
funding_alert_en_titanfall.wav      - Стиль Titanfall (английский)

--------------------------------------------------------------------
РЕКОМЕНДУЕМЫЙ ТЕКСТ ДЛЯ ЗАПИСИ:
--------------------------------------------------------------------

РУССКИЙ:
- "Фандинг превышен"
- "Внимание, фандинг"
- "Алерт фандинга"

АНГЛИЙСКИЙ:
- "Funding exceeded"
- "Funding alert"
- "Attention, funding"

--------------------------------------------------------------------
КАК ДОБАВИТЬ СВОЙ ГОЛОС:
--------------------------------------------------------------------

1. Запишите аудиофайл с нужной фразой
2. Сохраните его в эту папку (Sounds/Voice/)
3. Перезапустите программу
4. Новый голос появится в настройках фандинга
